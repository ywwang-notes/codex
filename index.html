作業二：倒數計時器

作業內容已整理為網頁文字，方便在 GitHub Pages 上直接閱讀。

任務流程

  1. 到 ChatGPT 點 Codex，依指示下載並安裝在自己的電腦。
  2. 到 GitHub，用 Google 帳號建立新帳號。
  3. 到 Codex，用自然語言做一個倒數計時器，並推送到 GitHub。
  4. 在 GitHub 手動建立新的 repository，從 repo 複製連結並貼到 Codex。
  5. 請 Codex 設定 GitHub Pages，讓作品顯示到網站上。
